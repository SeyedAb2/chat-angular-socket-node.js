# chat romm
### Angular 13 + socket.io + node.js
#### installation

##### run local server
```
cd backend
npm i
```
after add node_modules folder
```
node index.js
```
open new terminal 

##### run project
```
cd ..
cd frontend
npm i
```
after add node_modules folder
```
ng serve 
```

after served, you can see project [here](https://localhost:4200)
